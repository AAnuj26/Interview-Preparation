Blockquote title

The <blockquote> element is used to define a section of text that is a quotation from another source.
It's commonly used to display quotes or excerpts from articles, books, or other sources within a webpage.

Description List Title

The <dl> element is used to define a description list, which consists of a list of terms and their corresponding definitions.
The <dt> element is used within a <dl> element to define the term or name, which is followed by a corresponding definition in a <dd> element.
It's used when you want to present information in a term-definition format, such as glossaries or dictionaries.

Description Term 1 (<dt>)
Description Definition 1 (<dd>)
Description Term 2 (<dt>)
Description Definition 2 (<dd>)

    The <menu> element is a semantic alternative to the <ul> tag.
    It represents a list of commands or options. It's used to create navigational menus or lists of commands in an application.

    ol Title

    The <ol> element is used to define an ordered list, where each list item is automatically numbered.
    It's used when you have a list of items that need to be presented in a specific, sequential order.

      Ordered List
      1. First Item
      2. Second Item
      3. Third Item

    p Title

    The <p>element is used to define a paragraph of text.
    It's used to group together a block of text, providing visual separation and making the content more readable.

    pre Title

                    The <p> element is used to represent preformatted text. It's typically rendered in the web browser using a monospace font.
    
               It's used when you want to display code or text that should maintain its whitespace and formatting.

    hr title

    The <hr> element is used to create a horizontal rule or line, which can be used to separate content within a webpage.
    It's commonly used to visually divide sections of content or provide a clear visual break.
