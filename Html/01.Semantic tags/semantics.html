Header Title

The <header> tag represents introductory content or a group of introductory content. It often contains a group of navigation links or a site's branding information.

Nav Title

The <nav> tag represents a section of the document intended for navigation links. It's used to define a set of navigation links.

ul Title

The <ul> tag creates an unordered list, typically used for presenting a list of items, such as navigation links in this case.

li Title

The <li> tag defines a list item within an unordered list. In this context, each list item contains a navigation link.

Main Title

The <main> tag specifies the main content of a section or the web page. Enclose the primary content of a section or the entire page within <main> to indicate the most significant content.

Article Title

The <article> tag is used to define an independent, self-contained piece of content on a page. It can be an article, blog post, comment, or similar content.

Aside Section

The <aside> tag represents content that is related to the main content but can be considered separate from it. It might contain additional information, sidebars, or tangentially related content.

Section Title

The <section> tag defines a thematic grouping of content, typically containing related content such as chapters, headings, or thematic breaks.

Details title

Summary tag

The <details> tag defines a collapsed section of content that can be expanded if the user wishes to view it.

The <summary> tag specifies the summary or caption of a <details> element.

Footer title

The <footer> tag represents a footer for its nearest section or the entire body. It typically contains information such as copyright notices, contact information, or links to related documents.
